Full Application Architecture

This diagram illustrates the detailed architecture and flow of the incident management application, from the user chatbox through the backend, model, tool server (MCP), and back to the user. It shows how user input is processed, how intent and context are handled, how tool calls are made, and how results are returned and displayed.
